Wer wir sind:

Die Contabo GmbH bietet professionelle Hosting-Lösungen für alle Bedürfnisse - egal ob geschäftlich oder privat, egal ob für Internet-Einsteiger oder Experten mit höchsten Webhosting-Ansprüchen. Unsere Produktpalette umfasst das klassische Webhosting zu Discount-Preisen. Rootserver (Dedicated Server), virtuelle Server, Webspace und Colocation sind in allen erdenklichen Variationen und Größen bei uns erhältlich. Zudem steht unser Support an 365 Tagen im Jahr kostenlos zur Verfügung. Alle Server und Webspace-Pakete bei Contabo sind mit unserem kostenfreien DDoS-Schutz ausgestattet.[mehr]

Unsere Webhosting Angebote können sich mehr als sehen lassen: So ist unser Webspace-Paket M bereits ab 2,99 EUR erhältlich und bietet alles für die Erstellung der ersten eigenen Homepage. Es sind nahezu alle populären Skripte wie zum Beispiel WordPress, Joomla, Typo3, Drupal, phpBB, MediaWiki, Magento mit nur einem einzigen Mauseklick verfügbar. Sie können aus über 300 Web-Applikationen Ihre gewünschte Anwendung auswählen und in nur einem Schritt auf Ihrem Webspace Paket installieren. Als einer der günstigsten Webspace Anbieter für PHP und MySQL Hosting haben wir uns mittlerweile schon einen beachtlichen Namen auf dem Hosting-Markt gemacht.

Rootserver Power zum Webspace Paketpreis zeichnet unsere Virtual Private Server (VPS) aus. Unsere vServer können Sie bereits ab 7,99 EUR mieten. Entscheiden Sie sich schon heute für die nächste Generation unserer virtuellen Server, ausgestattet mit reinen SSD Festplatten für schnellstmögliches Arbeiten. Rechenaufwändige und speicherintensive Aufgaben bewältigen diese Server problemlos. Auch unsere bisherigen Modelle mit herkömmlichen HDD Festplatten und eingebauter SSD-Beschleunigung stehen Ihnen weiter zu Auswahl. Bei dieser Art von Server Virtualisierung steht Ihnen außerdem eine breite Auswahl an Linux- und Windows-Betriebssystemen zur Verfügung. Beispielsweise bieten wir die populären Linux Systeme Ubuntu, Debian, CentOS und Suse für Ihren virtuellen Server an. Alternativ entscheiden Sie sich für Windows Server 2016 oder Windows Server 2012. VServer mieten war für Sie noch nie zuvor so einfach und so günstig wie heute. Hier gelangen Sie zum vServer Vergleich auf Contabo.

Bei unseren Dedicated Servern setzen wir auf modernste und leistungsstarke Hardware. Mieten Sie Ihren eigenen Rootserver, wahlweise mit Linux oder Windows, bereits ab 49,99 EUR/ Monat. Aufgrund individueller Konfigurationsmöglichkeiten und Upgrades, wie z.B. der Installation von cPanel oder Plesk, steht Ihrem Server Hosting nichts mehr im Weg.

Entscheiden Sie sich für Colocation, so profitieren Sie von unserer preisgekrönten Rechenzentrumsinfrastruktur und der kosteneffizienten Unterbringung Ihrer eigenen Server-Hardware in einem unserer zwei Rechenzentren. Sie haben die Wahl zwischen unserem Rechenzentrum München und unserem Rechenzentrum Nürnberg. Server Housing Module (Server Racks) können Sie bereits ab 29,99 EUR pro Monat mieten. [weniger]
